MCP Setup

kdsnr-ai MCP 연결 안내 페이지입니다.
연결·인증은 최초 1회만 진행합니다.

ChatGPT

1. Dashboard에서 사용 승인 받기

좌측 사이드바의 Dashboard를 열고 Google 계정으로 로그인합니다.

  1. 소속과 인적사항을 입력해 승인 요청을 제출합니다.
  2. 관리자 승인이 완료될 때까지 기다립니다.
  3. 승인된 Google 계정을 이후 ChatGPT 연결에도 그대로 사용합니다.
  • Dashboard 승인 계정과 ChatGPT 플러그인 로그인 계정은 같아야 합니다.
  • 승인 과정에서 문제가 있을 경우 kdsnrai@gmail.com으로 인적사항 및 소속과 함께 문의 부탁드립니다.

2. 개발자 모드 켜기

ChatGPT 좌측 아래의 프로필에서 설정을 엽니다. 플러그인 → 개발자 모드로 이동한 뒤 개발자 모드를 켭니다.

3. 플러그인 스토어 열기

설정을 닫고 ChatGPT 좌측 사이드바에서 플러그인을 선택합니다.

4. 새 플러그인 추가하기

플러그인 스토어 우측 상단의 + 버튼을 누릅니다.

5. 플러그인 정보 입력하기

새 플러그인 화면에서 아래 항목을 순서대로 설정합니다.

항목입력값설명
아이콘선택 사항아래에서 내려받은 PNG 파일을 사용할 수 있습니다.
이름kdsnr-aiChatGPT의 플러그인 목록에 표시되는 이름입니다.
설명선택 사항입력하지 않아도 됩니다.
연결서버 URL터널이 아닌 서버 URL을 선택합니다.
인증OAuth고급 OAuth 설정은 변경하지 않습니다.

서버 URL 입력란에는 아래 주소를 그대로 붙여 넣습니다. 마지막으로 위험성 안내 아래의 확인란을 선택하고 만들기를 누릅니다.

6. Google 계정으로 로그인하기

kdsnr-ai 계정으로 로그인을 누른 뒤 ChatGPT 계정 접근을 허용합니다.

Dashboard에서 관리자 승인을 받은 것과 반드시 같은 Google 계정을 선택합니다.

7. 등록 확인하기

플러그인 상세 화면에 연결됨과 로그인한 계정이 표시되면 등록이 완료된 것입니다.

8. 사용하기

대화에 kdsnr-ai 추가

새 대화의 입력창에서 +를 누르고 kdsnr-ai를 선택합니다. 이후 문항 읽기나 HWPX 파일 생성을 평소와 같이 요청합니다.

도구 실행 허용

ChatGPT가 kdsnr-ai 기능을 실행하기 전에 확인창을 표시합니다. 한 번만 실행하려면 이번만 허용, 이후 같은 확인을 생략하려면 항상 허용을 선택합니다.

결과 파일 확인

파일 생성이 끝나면 대화창에 HWPX 파일이 표시됩니다. 파일을 내려받아 한글에서 열면 문장과 수식을 편집할 수 있습니다.
