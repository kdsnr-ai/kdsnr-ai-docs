Installation & Start

설치

pip install kdsnr-ai

Python 3.10 이상.

API Key 발급

API Key는 Dashboard에서 직접 발급합니다.

  1. 좌측 탭에서 Dashboard를 클릭하여 계정별 대쉬보드로 이동합니다.
  2. 구글 계정으로 로그인한 후, 소속과 인적사항을 적고 승인 요청을 제출합니다.
    * 최초 로그인에만 승인이 필요합니다. 승인은 별도 담당자 소통 없이 자동으로 처리되며, 보통 하루 이내에 처리됩니다.
  3. 승인이 완료된 후 대쉬보드에서 API Key를 발급합니다.
발급이 어려운 경우 소속 및 인적사항과 함께 kdsnrai@gmail.com으로 문의해 주세요.

Client

from kdsnr_ai import Client

client = Client(
    api_key="YOUR_API_KEY",
    timeout=120.0,
    max_retries=3,
    poll_interval=2.0,
)
파라미터기본값설명
api_key—발급받은 API 키. Authorization: Bearer로 전송.
timeout120.0HTTP 요청 타임아웃(초).
max_retries3일시 오류(429 · 502 · 503 · 504, 전송 오류) 재시도 횟수. 지수 백오프.
poll_interval2.0비동기 잡 폴링 간격(초).
