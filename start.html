Installation & Start

설치

pip install kdsnr-ai

Python 3.10 이상.

API Key 발급

현재 kdsnr-ai 라이브러리의 API Key는 담당자가 수동 발급·관리 중입니다.
발급 문의는 소속 및 인적사항과 함께 kdsnrai@gmail.com으로 요청 부탁드립니다.

Client

from kdsnr_ai import Client

client = Client(
    api_key="YOUR_API_KEY",
    timeout=120.0,
    max_retries=3,
    poll_interval=2.0,
)
파라미터기본값설명
api_key—발급받은 API 키. Authorization: Bearer로 전송.
timeout120.0HTTP 요청 타임아웃(초).
max_retries3일시 오류(429 · 502 · 503 · 504, 전송 오류) 재시도 횟수. 지수 백오프.
poll_interval2.0비동기 잡 폴링 간격(초).
